Nyle.pixel?(x, y, color)

概要

指定した座標の色が指定した色情報と一致するかどうかを調べます。

引数

引数名内容備考デフォルト値
xx座標
yy座標
color色情報
(「#RRGGBB」形式の色を表すコード
 または色を表すシンボル)		色に関する情報を参照

戻値

色情報の一致判定結果true  … 色情報が一致する
false … 色情報が一致しない

サンプル


def s.draw
  # マウスポインタの位置の色が緑(:GREEN)の場合に trueを表示します
  Nyle.draw_rect(50, 50, 100, 100, {color: :GREEN, fill: true})
  result = Nyle.pixel?(Nyle.mouse_x, Nyle.mouse_y, :GREEN)
  Nyle.draw_text(10, 30, result)
end
