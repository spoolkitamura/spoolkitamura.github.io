Nyle::Screen#start(interval=16)

概要

スクリーンの描画更新処理を開始します。

引数

引数名内容備考デフォルト値
intervalスクリーンの描画間隔
(ミリ秒)		１秒(1,000ミリ秒)間に約60コマの描画をおこなうために
intervalのデフォルト値を16に設定		16

戻値

なし

サンプル


s = Nyle.create_screen

def s.setup
  @x = 100
end

def s.draw
  Nyle.draw_circle(@x, 240, 100)
end

def s.update
  @x += 10
end

# １秒間隔でスクリーンの描画が更新されます
s.start(1000)
