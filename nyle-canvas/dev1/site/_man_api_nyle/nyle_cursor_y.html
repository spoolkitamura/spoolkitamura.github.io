Nyle.cursor_y

概要

カーソルキーの上下キーの入力をY座標の増分(-1, 0, 1のいずれか)として返します。

引数

なし

戻値

Y座標の増分-1 … 上カーソルキーが押された
 0 … 上下いずれのカーソルキーも押されていない
 1 … 下カーソルキーが押された

サンプル


def s.setup
  @x = 320
  @y = 240
end

def s.draw
  # カーソルキーによって円の描画位置が動きます
  @x += Nyle.cursor_x
  @y += Nyle.cursor_y
  Nyle.draw_circle(@x, @y, 50)
end
