Nyle.key_release?(key_code)

概要

キーボードの指定のキーが離された瞬間だけ真を返します。

引数

引数名内容備考デフォルト値
key_codeキーコードに関する情報を参照

戻値

キーの状態true  … 指定のキーが離された瞬間である
false … 指定のキーが離された瞬間ではない

サンプル


def s.setup
  @color = :BLUE
end

def s.draw
  # スペースキーが離された瞬間に円の色を変更します
  if Nyle.key_release?(:SPACE)
    if @color == :BLUE
      @color = :CYAN
    else
      @color = :BLUE
    end
  end
  Nyle.draw_circle(320, 240, 100, {color: @color, fill: true})
end
