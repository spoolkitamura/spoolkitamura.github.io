Nyle.load_image(image_name, option={})

概要

指定された画像データを読み込みます。

画像データは、あらかじめ Nyle-canvasのエディタ上に
ドラッグアンドドロップしておいてください。

引数

引数名内容備考デフォルト値
image_name画像データ名

オプション(option)

オプション名内容備考デフォルト値
:color_key透過色色を表すシンボルまたは「#RRGGBB」形式のコード
(色に関する情報を参照)
※実装途中のため :WHITE限定対応		:WHITE
:sx拡大率(X方向)0.1以上の値1.0
:sy拡大率(Y方向)0.1以上の値1.0
:cx抽出範囲の左上X座標nil
:cy抽出範囲の左上Y座標nil
:cw抽出範囲の幅nil
:ch抽出範囲の高さnil

戻値

Imageクラスのオブジェクト

サンプル


def s.setup
  # 画像データ "picture.png"を読み込み、
  # 縦横のサイズを 70%に縮小します
  @image1 = Nyle.load_image("picture.png", {sx: 0.7, sy: 0.7})

  # 画像データ "picture.png"を読み込み、
  # 座標(10, 10)を基点にして100×100の範囲を抽出します
  @image2 = Nyle.load_image("picture.png", {cx: 10, cy: 10, cw: 100, ch: 100})
end

def s.draw
  Nyle.draw_image(10,  10, @image1)
  Nyle.draw_image(10, 120, @image2)
end
