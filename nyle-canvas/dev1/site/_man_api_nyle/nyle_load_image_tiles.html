Nyle.load_image_tiles(image_name, m, n, option={})

概要

指定された画像データを読み込んで m×n個の要素に分割します。

画像データは、あらかじめ Nyle-canvasのエディタ上に
ドラッグアンドドロップしておいてください。

引数

引数名内容備考デフォルト値
image_name画像データ名
m横方向の分割数
n縦方向の分割数

オプション(option)

オプション名内容備考デフォルト値
:color_key透過色色を表すシンボルまたは「#RRGGBB」形式のコード
(色に関する情報を参照)
※実装途中のため :WHITE限定対応		:WHITE
:sx拡大率(X方向)0.1以上の値1.0
:sy拡大率(Y方向)0.1以上の値1.0

戻値

Imageクラスのオブジェクトを収録した２次元配列

サンプル


def s.setup
  # 画像データ "picture.png"を読み込み、
  # 横3×縦4個に分割して 120%に拡大した計12個のオブジェクトを
  # 2次元配列として収録します
  @image = Nyle.load_image_tiles("n_i_chara.png", 3, 4, {sx: 1.2, sy: 1.2})
end

def s.draw
  3.times do |i|
    4.times do |j|
      Nyle.draw_image(10 + 40 * i,  10 + 40 * j, @image[i][j])
    end
  end
end
