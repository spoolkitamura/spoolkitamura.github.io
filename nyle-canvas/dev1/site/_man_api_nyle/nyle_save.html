Nyle.save

概要

現在の座標系の状態などを保持し、ブロックを抜けるときにその状態を復元します。

引数

引数名内容備考デフォルト値
{}実行されるブロック

戻値

なし

サンプル


def s.draw
  # 座標系操作の範囲を限定します
  Nyle.save do
    Nyle.translate(50, 50)              # 座標系の原点座標を (50, 50)に移動
    Nyle.rotate(Math::PI / 4)           # 座標系を 45度回転
    Nyle.scale(3, 3)                    # 座標系を縦横それぞれ 3倍
    Nyle.draw_line(10, 10, 100, 10)     # 直線を描画(上記の変換座標系で描画)
  end
  Nyle.draw_line(10, 10, 100, 10)       # 直線を描画(通常の座標系で描画)
end
