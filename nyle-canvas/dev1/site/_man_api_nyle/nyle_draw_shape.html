Nyle.draw_shape(points, option={})

概要

複数の頂点座標を直線で結んで図形を描きます。

引数

引数名内容備考デフォルト値
points頂点座標[x, y]の配列

オプション(option)

オプション名内容備考デフォルト値
:weight線の太さ1以上の整数値2
:color描画色色を表すシンボルまたは「#RRGGBB」形式のコード
(色に関する情報を参照)		:BLACK
:a描画色のα値
(透明度)		0.0～1.01.0
:fill塗りつぶしtrue   矩形を塗りつぶす
false  矩形を塗りつぶさない		false
:close閉鎖図形
(始点と終点の結合)		true   閉鎖図形にする
false  閉鎖図形にしない		false
:cap先端の形状:BUTT     通常
:ROUND  丸み
:SQUARE  角		:BUTT

戻値

なし

サンプル


def s.setup
  @points = [[100, 50], [200, 50], [180, 170]]
end

def s.draw
  # 座標(100, 50), (200, 50), (180, 170)を直線で結んで囲まれた青い図形を描きます。
  Nyle.draw_shape(@points, {color: :BLUE, close: true, weight: 5})
end
