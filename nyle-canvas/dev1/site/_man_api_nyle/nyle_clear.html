Nyle.clear

概要

キャンバス上の描画内容をクリアします。

Nyle::Screenの trace属性が trueの場合に
描画されたキャンバス上の内容をクリアすることができます。
(ただし、bgcolorで指定された背景色はそのまま残ります)

Nyle::Screenの trace属性が falseの場合には
描画のたびにキャンバスが毎回書き換えられているので
clearによる効果はとくにありません。

引数

なし

戻値

なし

サンプル


# キャンバスの trace属性を trueに設定
s = Nyle.create_screen(640, 480, {bgcolor: :WHITE, trace: true})

def s.draw
  if Nyle.mouse_down?(:L)
    # マウスの左ボタンを押しながら動かすと線が描画されます
    Nyle.draw_circle(Nyle.mouse_x, Nyle.mouse_y, 5, {fill: true})
  end
  if Nyle.mouse_release?(:R)
    # マウスの右ボタンをクリックするとキャンバスがクリアされます
    Nyle.clear
  end
end
