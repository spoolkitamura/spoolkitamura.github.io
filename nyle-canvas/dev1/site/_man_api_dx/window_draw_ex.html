Window.draw_ex(x, y, image, option={})

概要

Image.load()メソッドまたは Image.load_tiles()メソッドでロードした画像に対して
拡大率および回転角度を指定してキャンバスに描画します。

引数

引数名内容備考デフォルト値
xX座標
yY座標
image画像オブジェクト

オプション(option)

オプション名内容備考デフォルト値
:scale_xX方向(横方向)の拡大率0より大きな値1
:scale_yY方向(縦方向)の拡大率0より大きな値1
:angle回転角度0を基準として時計回りに360で一周0
:center_x拡大する際の中心となるX座標画像の左上を原点とする相対座標
(nilの場合は画像の中央)		nil
:center_y拡大する際の中心となるY座標画像の左上を原点とする相対座標
(nilの場合は画像の中央)		nil
:alpha透明度(α値)0～255255

戻値

なし

サンプル


img = Image.load('logo.png')
Window.loop do
  # 画像の中央を基準にして
  # 横方向に2.0倍、縦方向に0.5倍に拡大した上で
  # 45度回転させて半透明(α=128)で描画します
  Window.draw_ex(320, 240, img, {scale_x: 2.0, scale_y: 0.5, angle: 45, alpha: 128})
end
