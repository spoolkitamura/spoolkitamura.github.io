Image#set_color_key(color) 【未実装】

概要

画像データについて、指定された色と同じ色を透明(α値＝0)にします。

【注意】
    内部処理がまだ実装されていないため、
    このメソッドを実行しても指定色は透明化されません。

    その代わりに、Image.loadおよび Image.load_tilesメソッド内で
    暫定的に白色を自動的に透明にするようになっています。

引数

引数名内容備考デフォルト値
color透過指定色[A, R, G, B]形式の色配列
(色に関する情報を参照)

戻値

Imageオブジェクト

サンプル


# 画像を読み込んで、白色の部分を透明にしてキャンバスに描画します
img = Image.load("logo.png")
img.set_color_key(DX::C_WHITE)   # 未実装
Window.loop do
  Window.draw(320, 240, img)
end
