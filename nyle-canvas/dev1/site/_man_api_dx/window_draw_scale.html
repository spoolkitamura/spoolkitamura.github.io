Window.draw_scale(x, y, image, scale_x, scale_y, center_x=nil, center_y=nil)

概要

Image.load()メソッドまたは Image.load_tiles()メソッドでロードした画像に対して
縦、横の拡大率を指定してキャンバスに描画します。

引数

引数名内容備考デフォルト値
xX座標
yY座標
image画像オブジェクト
scale_xX方向(横方向)の拡大率0より大きな値
scale_yY方向(縦方向)の拡大率0より大きな値
center_x拡大する際の中心となるX座標画像の左上を原点とする相対座標
(nilの場合は画像の中央)		nil
center_y拡大する際の中心となるY座標画像の左上を原点とする相対座標
(nilの場合は画像の中央)		nil

戻値

なし

サンプル


img = Image.load('logo.png')
Window.loop do
  # 画像の中央を基準にして
  # 横方向に2.0倍、縦方向に0.5倍に拡大して描画します
  Window.draw_scale(320, 240, img, 2.0, 0.5)
end
