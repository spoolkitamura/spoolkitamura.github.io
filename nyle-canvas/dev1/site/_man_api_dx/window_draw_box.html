Window.draw_box(x1, y1, x2, y2, color)

概要

キャンバス上の座標(x1, y1)と座標(x2, y2)を対角とする矩形を描画します。

引数

引数名内容備考デフォルト値
x1左上X座標
y1左上Y座標
x2右下X座標
y2右下Y座標
color描画色[A, R, G, B]形式の色配列
(色に関する情報を参照)

戻値

なし

サンプル


Window.loop do
  # 座標(100, 100)と座標(540, 380)を対角とする黄色い矩形を描画します
  Window.draw_box(100, 100, 540, 380, DX::C_YELLOW)
end
