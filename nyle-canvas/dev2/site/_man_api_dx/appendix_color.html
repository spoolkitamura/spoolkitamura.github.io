色に関する情報

仕様

[A, R, G, B]の形式の配列(＝色配列)で色の指定をおこないます。
それぞれの要素の内容は下記のとおりです。

要素内容備考値の範囲
Aα値透明度を表す
(0は完全な透明、255は完全な不透明)		0～255
RREDの値0～255
GBLUEの値0～255
BGREENの値0～255

定数

Nyle-canvas(DXRubyスタイル)では、下記の色配列が定数として定義されています。

定数名値色
C_BLACK[255, 0, 0, 0]黒
C_RED[255, 255, 0, 0]赤
C_GREEN[255, 0, 255, 0]緑
C_BLUE[255, 0, 0, 255]青
C_YELLOW[255, 255, 255, 0]黄
C_CYAN[255, 0, 255, 255]シアン
C_MAGENTA[255, 255, 0, 255]マゼンダ
C_WHITE[255, 255, 255, 255]白
