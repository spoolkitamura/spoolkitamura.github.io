Input.x

概要

カーソルキーの左右キーの入力をX座標の増分(-1, 0, 1のいずれか)として返します。

引数

なし

戻値

X座標の増分-1 … 左カーソルキーが押された
 0 … 左右いずれのカーソルキーも押されていない
 1 … 右カーソルキーが押された

サンプル


include DX
x = 320
y = 240
Window.loop do
  # カーソルキーによって円の描画位置が動きます
  x += Input.x
  y += Input.y
  Window.draw_circle(x, y, 50, C_GREEN)
end
