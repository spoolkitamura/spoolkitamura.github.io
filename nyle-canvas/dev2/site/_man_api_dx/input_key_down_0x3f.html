Input.key_down?(key_code)

概要

キーボードの指定のキーが押されているあいだ真を返します。

引数

引数名内容備考デフォルト値
key_codeキーコードに関する情報を参照

戻値

キーの状態true  … 指定のキーが押されている
false … 指定のキーが押されていない

サンプル


include DX
Window.loop do
  # スペースキーが押されている間だけキャンバスに円を描画します
  if Input.key_down?(K_SPACE)
    Window.draw_circle_fill(320, 240, 100, C_YELLOW)
  end
end
