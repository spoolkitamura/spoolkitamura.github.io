Window.draw(x, y, image)

概要

Image.load()メソッドまたは Image.load_tiles()メソッドでロードした画像を
キャンバスに描画します。

引数

引数名内容備考デフォルト値
xX座標
yY座標
image画像オブジェクト

戻値

なし

サンプル


img = Image.load('logo.png')
Window.loop do
  # 画像を描画します
  Window.draw(320, 240, img)
end
