Nyle.pixel(x, y)

概要

指定した座標の色情報を取得します。

引数

引数名内容備考デフォルト値
xx座標
yy座標

戻値

色情報
(「#RRGGBB」形式の色を表すコード)		色に関する情報を参照

戻値


サンプル


def s.draw
  # マウスポインタの位置の色情報を表示します
  Nyle.draw_rect(50, 50, 100, 100, {color: :GREEN, fill: true})
  color = Nyle.pixel(Nyle.mouse_x, Nyle.mouse_y)
  Nyle.draw_text(10, 30, color)
end
