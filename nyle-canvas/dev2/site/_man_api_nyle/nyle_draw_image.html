Nyle.draw_image(x, y, image, option={})

概要

基点座標に画像を描画します。

引数

引数名内容備考デフォルト値
x基点のX座標
y基点のY座標
imageImageクラスのオブジェクトNyle.load_image または
Nyle.load_image_tiles メソッドで
読み込まれたオブジェクト

オプション(option)

オプション名内容備考デフォルト値
:pos基点のポジション:CORNER     imageの左上座標
:CENTER     imageの中心座標		:CORNER
:a画像描画時のα値
(透明度)		0.0～1.01.0

戻値

Imageクラスのオブジェクト

サンプル


def s.setup
  # 画像データ "picture.png"の読み込み
  @image = Nyle.load_image("picture.png")
end

def s.draw
  # 画像の中心が座標(320, 240)になるように半透明で描画します
  Nyle.draw_image(320, 240, @image, {pos: :CENTER, a: 0.5})
end
