Nyle.draw_circle(x, y, r, option={})

概要

中心の座標と半径を指定して円を描きます。

引数

引数名内容備考デフォルト値
x円の中心のX座標
y円の中心のY座標
r円の半径

オプション(option)

オプション名内容備考デフォルト値
:weight線の太さ1以上の整数値2
:color描画色色を表すシンボルまたは「#RRGGBB形式」コード
(色に関する情報を参照)		:BLACK
:a描画色のα値
(透明度)		0.0～1.01.0
:fill塗りつぶしtrue   矩形を塗りつぶす
false  矩形を塗りつぶさない		false

戻値

なし

サンプル


def s.draw
  # 座標(320, 240)を中心とした半径100の黄色い円を描画して塗りつぶします
  Nyle.draw_circle(320, 240, 100, {color: :YELLOW, fill: true})
end
